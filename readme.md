# Build From Source

On Windows:

- Vulkan SDK 1.1
- CMake 3.10
- Git

clone the branch PrecomputeAtmosphericScattering

~~~
git clone -b PrecomputeAtmosphericScattering https://github.com/hanayangl5/horizon.git
~~~

use CMake to generate solution file

~~~
cmake -D build .
~~~

build and run `example/atmosphere`
